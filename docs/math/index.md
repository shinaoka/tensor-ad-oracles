# Math Notes

`tensor-ad-oracles` publishes two first-class artifacts:

- mathematical AD notes
- the machine-readable oracle database

The mathematical notes live under `docs/math/`. Standalone linalg operations are
documented as one note per operation, while shared scalar formulas are grouped
where that keeps the corpus easier to maintain.

Published DB families are linked to the note corpus through
`docs/math/registry.json`.
